== Description ==

Create and manage beautiful, interactive team showcases with our drag and drop builder and shortcode generator.
The plugin is built with developers in mind. Make advanced customisations to the frontend design and options without touching the plugins code.

[Documentation](https://team-showcase.awesome-plugins.com)  |  [Demo](https://awesome-plugins.com/team-showcase-demo-page/) | [Download Premium Beta Plugin](https://awesome-plugins.com/team-showcase-beta-plugin/) | Premium coming soon

**Features**

  **5 layout options**
   Intuitive skin function with collapse, modal grids, list view, table view and carousel.

   **Unlimited Teams and Members**
   No limits in adding teams or members in the free version

  **Drag and Drop interface**
 Manage team members easily with a drag and drop interface.

 **Extra content options**
Present extra content in an expanding grid, layover or link to the member post.

**Easy management using Shortcodes**

Easily add the team showcase or team member to any post by using the generated shortcode.

**Compatible with any theme**

We tested it in many themes without any problems, we use the absolute minimum css so it is easy to finetune the accordion in your theme or by using the customizer

== Installation ==

Activate the team showcase plugin from the plugin interface by downloading it directly or uploading the zip file. After activation a new menu item “Team showcase” should be visible.

== Frequently Asked Questions ==

= Can I add team showcases to a page/post ? =

Yes, no problem you can easily add multiple shortcodes to a post

= I need more features can I get them =

The premium version is coming soon.  But you can test our beta version. [Download Premium Beta Plugin](https://awesome-plugins.com/team-showcase-beta-plugin/)

= Can I customize the colors/style of the team showcase ? =

Yes, but you need some CSS knowledge, you can easily do that by adding classes to your theme or using the customizer.

= Is the team showcase responsive ? =

Yes it works fine in all screen sizes and devices

= Can I translate the plugin in my language? =

Yes the plugin is 100% translation ready

= Does this plugin work with Gutenberg ? =

Yes, you can paste the accordion shortcode into a ‘Paragraph Block’ or use the ‘Shortcode Block’.

= How can I get support? =

If you face any problem with our plugin or something is not working as it should, first follow the preliminary steps:
Test the plugin with a WordPress default theme, to be sure that the error is not caused by the theme you are currently using.

Deactivate all plugins you are using and check if the problem is still occurring.

If you still have problem you can visit our  [documentation](https://team-showcase.awesome-plugins.com) to see if you can find answers there

If none of the previously listed actions helps you solve the problem, then, submit a ticket in the forum and describe your problem accurately, specify WordPress and plugin versions you are using and any other information that might help us solve your problem as quickly as possible.

= I found a bug =

If you find a bug, please report it in our active support forum. You will be replied once the issue is resolved.

= how can I support this plugin? =

It's a free version so we don't need your money, but you can write a nice review in the review section.
